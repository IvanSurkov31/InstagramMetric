InstagramMetric

Программный комплекс - автоматизированного мониторинга постов, опубликованных в 
сообществах социальной сетях Instagram. Комплекс осуществляет сканирование данных
ресурсов, а затем мониторинг текстового контента в течение времени его жизни, включая анализ ряда метрик.

Инструкция:
1. В файл NamePublic.txt заносим название сканируемых пабликов.
2. В файл Keywords.txt заносим ключевые слова на исследуемую тематику.
